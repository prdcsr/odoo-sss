Cancel or Confirm Multiple Journal Entries

This module helps to Cancel or Confirm multiple journal entries from the journal entries tree view.

Part-01 : For Cancel Journal Entries

Step 1:

In Accounting -> select a journal entries to Cancel those entries.

Step 2:

In Accounting -> Click on "Action" and select "Cancel journal entries".

Step 3:

In Accounting -> now open a wizard, select "Cancel journal entries" button to Cancel journal entries.

Step 4:

In Accounting -> Your Cancel journal entries show in tree view.

Part-02 : For Confirm Journal Entries

Step 1:

In Accounting -> select a journal entries to confirm those entries.

Step 2:

In Accounting -> Click on "Action" and select "Confirm journal entries".

Step 3:

In Accounting -> now open a wizard, select "Confirm journal entries" button to Confirm journal entries.

Step 4:

In Accounting -> Your Confirm journal entries show in tree view.
